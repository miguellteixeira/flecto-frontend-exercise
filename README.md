A simple landing page to see all the planets from Star Wars.

## Tech Stack

- React 18
- Next 14
- Tailwind CSS
- TanStack Query
- Ky
- Zustand
- Flowbite
- Typescript
- Vitest

## Usage

Install dependencies

```bash
yarn install
```

### Development

Run

```bash
yarn dev
```

Then visit http://localhost:3000

### Linting

Run

```bash
yarn lint
```

### Testing

Run

```bash
yarn test
```

### Build

```bash
yarn build
```
